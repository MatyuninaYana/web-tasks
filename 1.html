Задание 1

Задание заключается в верстке макета для десктопных устройств на тему «Междугородний трансфер по Крыму».

Макет содержит несколько секций: шапку, баннер, форму для заявки на заказ трансфера и список тарифов.

Ваша задача — сверстать только десктопную версию макета.

Прилагаемые файлы

Для выполнения задания необходимо скачать архив с дополнительными материалами web2024.zip и распаковать его.

Дальшейшую работу над заданием необходимо проводить только в файлах index.html и css/style.css

Макеты




Задачи

  • Сверстать шапку
  • Сверстать баннер
  • Сверстать форму. Общая структуры формы уже есть в файле index.html, поэтому остаётся только добавить стили для элементов формы. Декомпозировать элементы можно
  • Сверстать тарифы
  • Подключить шрифты из папки fonts в style.css

Примечания

  • Атрибут data-test у тегов убирать нельзя
  • В файле css/style.css есть небольшой фрагмент кода. Если вам это нужно, вы можете воспользоваться им
  • Отправьте в систему zip-архив с решением задачи

Оценивание

Вы можете получить за это задание 15 баллов при условии реализации всей функциональности.
